# Agreed Purchase UI

Front-End for [Agreed Purchase]

## Dependencies
* Java 11
* Maven (Build)

## Run
```
mvn spring-boot:run
```

## Run with jar file
```
mvn clean install 

java -jar ./target/${PROJECT_NAME}.jar
```

## Initial page
```
http://localhost:8080/
```

To run the project locally, you need run before [agreed-purchase](https://github.com/isacaguiar/agreed-purchase).
